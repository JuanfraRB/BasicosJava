# Hello Java ☕

Primer programa básico en Java que muestra un mensaje en consola.

Este proyecto forma parte de una serie de pequeños ejemplos para practicar Java desde los conceptos más simples.

---

## 📌 Descripción

El programa imprime en consola el mensaje:

```
Hello Java
```

Se trata del ejemplo clásico utilizado para comprobar que el entorno de desarrollo y el compilador de Java funcionan correctamente.

---

## 📂 Estructura

```
src/hellojava/
│
─── HelloJava.java

```

---

## ▶️ Compilar el programa

```bash
javac HelloJava.java
```

---

## ▶️ Ejecutar el programa

```bash
java HelloJava
```

---

## 💻 Código

```java
public class HelloJava {

    public static void main(String[] args) {

        System.out.println("Hello Java!!");

    }

}
```

---

## 🎯 Objetivo

* Verificar que Java está correctamente instalado
* Entender la estructura básica de un programa Java
* Familiarizarse con el método `main`
