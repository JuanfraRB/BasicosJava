# ☕ Java desde Cero

Repositorio con una colección de pequeños programas en **Java** que comienzan desde lo más básico y van avanzando progresivamente hacia conceptos más complejos.

El objetivo es construir una base sólida repasando los fundamentos del lenguaje y desarrollando ejemplos claros y prácticos.

---

## 📚 Contenido del repositorio

Cada carpeta contiene un ejemplo independiente centrado en un concepto concreto de Java.

| Nº | Proyecto            | Concepto                        |
| -- | ------------------- | ------------------------------- |
| 01 | Hello Java          | Primer programa en Java         |
| 02 | Enum                | Tipo dato enum                  |
| 03 | Enum2               | Tipo dato enum, en otro archivo |
| 04 | CalculosAritmeticos | Funciones matemáticas           |
| 05 | Operaciones         | Tipo dato booleano              |

*(Se irán añadiendo nuevos ejemplos progresivamente.)*

## ▶️ Ejecutar un ejemplo

1. Acceder al directorio del proyecto

```bash
cd hellojava
```

2. Compilar el programa

```bash
javac HelloJava.java
```

3. Ejecutarlo

```bash
java HelloJava
```
